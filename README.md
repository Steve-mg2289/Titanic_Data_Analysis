# 🚢 Titanic Data Analysis (R)

## 📌 Project Overview
This project explores the Titanic dataset using R. The goal is to clean raw data, perform exploratory data analysis (EDA), and generate insights about passenger survival.

---

## 🧹 Data Cleaning
- Handled missing values (e.g., Age column had 177 missing values)
- Removed irrelevant columns (e.g., Cabin)
- Checked for duplicates
- Converted variables into appropriate data types

---

## 📊 Data Visualization
The following visualizations were created:
- Survival count
- Survival by gender
- Age distribution
- Survival by passenger class
- Age vs survival

---

## 🔍 Key Insights
- Females had higher survival rates than males
- First-class passengers had better survival chances
- Age had some influence on survival

---

## 🛠 Tools Used
- R
- ggplot2
- dplyr
- RStudio
